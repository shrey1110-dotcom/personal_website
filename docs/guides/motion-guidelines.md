# Motion Guidelines

Motion is treated as support for structure, not a feature by itself.

## Allowed motion

- fade-in reveals
- slight vertical lift
- low-amplitude hover movement
- slow ambient marquee motion

## Avoided motion

- large parallax swings
- rotating decorative elements
- cursor gimmicks
- heavy looping animations in primary reading areas

## Rule of thumb

If motion competes with the copy, the motion loses.
