# Interaction Primitives

The homepage uses a small set of reusable interaction ideas.

## Current primitives

- section reveal timing through Framer Motion
- smooth anchor scrolling through Lenis
- restrained tilt on major premium surfaces
- custom cursor for fine-pointer devices only
- low-contrast ambient information rails

## Rule

Any new interaction should reinforce hierarchy or tactility. If it exists only to look impressive, it should not ship.
