# Work Experience Showcase

The work experience section is designed as an interactive showcase rather than a static resume block.

## Goals

- Make the two strongest roles feel selected and deliberate.
- Let the user switch focus quickly.
- Keep company metadata visible without overwhelming the bullets.

## Interaction pattern

- Hover or focus on a company selector to preview it.
- Click to lock a role in place.
- Prev/next buttons provide a second path for navigation.

## Content priorities

- Company name
- Role and date
- Short description
- Concrete bullets
- Tech stack
- Company metadata

## Design rule

The left panel should communicate personal contribution.
The right panel should communicate company context.
