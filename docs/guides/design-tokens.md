# Design Tokens

The visual system uses a restrained dark palette with a single cool accent family.

## Surface tones

- background: deep navy-black
- panel: slightly lifted charcoal
- border: low-contrast cool gray
- text primary: near-white
- text secondary: muted slate

## Accent language

- primary accent: indigo / violet
- supporting accent: cyan-teal
- gradients are used sparingly for depth, not spectacle

## Typography

- Inter via `next/font/google`
- large headline contrast
- compact uppercase section labels
- readable body text at moderate measure
