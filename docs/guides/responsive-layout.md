# Responsive Layout Notes

The homepage is designed to keep hierarchy intact from mobile through large desktop.

## Mobile

- stacked sections
- reduced visual density
- tighter spacing
- project bands collapse into clean vertical cards

## Desktop

- wider content rails
- alternating project composition
- two-column hero with portrait panel
- stronger whitespace rhythm

## Constraint

No part of the layout should rely on hover or huge viewport height to make sense.
