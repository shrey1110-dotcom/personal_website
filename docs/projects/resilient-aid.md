# ResilientAid

ResilientAid is positioned on the homepage as a flagship product because it communicates systems thinking quickly.

## Core message

- Aid distribution should be traceable.
- Donors should be able to see where funds move.
- Beneficiaries and vendors need fast settlement, not admin friction.

## Homepage proof points

- `425 tx/s`
- `0%` platform fees
- `<2s` settlement time
- `100%` on-chain transparency

## Interfaces

- Admin Portal
- Donor Platform
- Beneficiary App
- Vendor POS

## Visual direction

- The demo window should look like an active logistics or payout system.
- The stats row is more important than decorative UI.
- The interface links reinforce that the product has distinct user surfaces.
