# VeriDegree

VeriDegree is one of the clearest examples of trust infrastructure on the portfolio.

## Core message

- Schools issue digital credentials once.
- Students hold them directly.
- Employers verify them quickly without manual back-and-forth.

## Homepage proof points

- Mainnet Algorand deployment
- Soulbound academic records
- Instant employer verification
- Private selective disclosure

## Visual direction

- The demo window should read like a verification flow, not a crypto dashboard.
- Verification status needs to be obvious within a second.
- The credential state is the important outcome, not blockchain jargon.
