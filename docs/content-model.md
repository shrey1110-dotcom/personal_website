# Content Model

Most of the site copy is driven from `lib/portfolio-content.ts`.

## Structured content

- navigation items
- hero chips
- hero stats
- ambient rails
- retain product copy, metrics, integrations, and callouts
- about build lines
- about blocks
- project entries, metrics, interfaces, and links
- skill groups
- contact links

## Benefit

Keeping content centralized makes the homepage easier to revise without rewriting layout code.
