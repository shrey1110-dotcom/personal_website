# Homepage Structure

The homepage is intentionally linear so recruiters can scan it quickly.

## Section order

1. Header
2. Hero
3. About
4. Projects
5. Featured Project
6. Skills
7. Contact
8. Footer

## Why this order

- The hero introduces the person before the work.
- Projects appear before the deep case-study treatment.
- Skills support the project evidence instead of leading the story.
- Contact closes immediately after the strongest proof.
