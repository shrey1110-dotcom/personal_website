# Asset Inventory

The homepage currently relies on a small, fixed public asset set.

## Files in `public/`

- `og.png`: social preview image
- `og-source.svg`: editable source for the social preview
- `retain-screenshot.png`: RETAIN AI dashboard proof image
- `retain-screenshot-source.svg`: editable source for the RETAIN AI visual fallback
- `resume.pdf`: resume download target

## Guidance

- Keep asset names stable because multiple links and metadata fields rely on them.
- Prefer updating files in place over renaming them.
- Recheck GitHub Pages paths after any asset replacement.
