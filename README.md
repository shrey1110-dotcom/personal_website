# Shreyansh Sharma Portfolio

Personal portfolio site built with Next.js 14 App Router, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```
